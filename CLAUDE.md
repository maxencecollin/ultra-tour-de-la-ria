# Règles de développement pour Claude

## Commits Git

- **Ne jamais ajouter de co-auteur** dans les commits (pas de `Co-Authored-By`)
- **Avant chaque commit** :
  1. Relire systématiquement tout le code modifié
  2. Vérifier la cohérence et la qualité du code
  3. Lancer les tests si disponibles
  4. S'assurer que le site s'affiche correctement (pas d'erreurs console)

## Maintenance du projet

- **Résumer régulièrement** la conversation et les décisions prises
- **Nettoyer les fichiers abandonnés** : si une approche est abandonnée, supprimer les fichiers/code inutilisés
- **Garder le projet propre** : pas de fichiers temporaires, pas de code commenté inutile

## Structure du site

- Site statique HTML/CSS/JS
- Images optimisées pour le web (max ~500Ko par image)
- Vidéos compressées (format web, max ~10Mo par vidéo)
